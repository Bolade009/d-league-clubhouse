How to Join D League Clubhouse
Full walkthrough for new managers
STEP 1 — Join the official FPL D League
  • The Commissioner creates a private league in Fantasy Premier League.
  • You will receive the league code or direct join link (usually via WhatsApp group).
  • Go to fantasy.premierleague.com
  • Create or log into your FPL account.
  • Join the league using the code.
  • Build your team before the deadline.
This is required — the Clubhouse pulls your real squad, captain picks, points from this FPL team.
STEP 2 — Get access to this Clubhouse app
The league admin (commissioner) will add you to the D League Clubhouse system and send you:
  • Your registered email
  • Your personal Access Code (e.g. yourname2026)
When requesting access, provide your FPL club name to confirm you joined the correct league.
STEP 3 — Login
Return to this page. Enter the email + access code and click ENTER THE CLUBHOUSE.
First time? The server must be running (or use the live link the group shares).
STEP 4 — Pay your fees
After logging in you’ll see Pay buttons:
  • FPL Season — full entry fee + admin (see current amounts in UI)
  • UCL Season — full entry fee + admin
Pay via Paystack. Only confirmed webhooks mark you as paid. You can pay one or both.
STEP 5 — Use & monitor your dashboard
  • Sync Scores regularly to pull latest FPL points.
  • LINEUP VIEWER — dropdown to see any manager’s real FPL squad + captain + chip + projected points. Bench shown below the pitch.
  • Standings — combined + FPL + UCL races. Click rows for full profile + recent picks.
  • Ledger — see transaction history, winnings, payouts.
  • H2H & Challenges — real matches from the group.
  • FPL League data — admin can load the official FPL league ID so H2H and standings match FPL’s own system.
Questions? Message the commissioner. Everything is transparent and recorded via Paystack webhooks + FPL public APIs.
